# 🐍 Snake Xenzia

A classic Snake game built with Python and Turtle Graphics.

Control the snake, eat food to grow longer, increase your score, and avoid colliding with the walls or your own body.

## 🎮 Features

- Smooth snake movement

- Food spawning at random locations

- Score tracking

- Snake grows after eating food

- Collision detection:

  - Wall collision

  - Self collision

- Game Over screen with final score

## 📸 Gameplay

The snake moves continuously across the screen.

- Eat the food to earn points.

- Each food item increases your score by 1.

- The snake grows longer every time it eats.

- The game ends if:

  - The snake hits a wall

  - The snake collides with itself

## 🛠️ Built With

- Python 3

- Turtle Graphics

## 🚀 Getting Started

### Prerequisites

Make sure Python 3 is installed:

```bash

python --version

```

### Run the Game

Clone the repository:

```bash

git clone https://github.com/jibinsajujoseph/snake-game.git

cd snake-game

```

Run:

```bash

python main.py

```
